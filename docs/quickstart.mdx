<Steps>
  <Step title="Install CLI">
  ```sh
  curl -fsSL "https://get.yomo.run" | sh
  ```
  </Step>
  <Step title="Initialize a LLM Function">
  ```sh
  yomo init tool-get-weather
  ```

an example llm function calling will be created in the `ool-get-weather` directory:

```sh

.rw-r--r--@ 266 c3ylabs 15 Apr 10:00 └── tsconfig.json
```

  </Step>
  <Step title="Run YoMo Server and expose MCP server">
```sh
yomo serve
```

The output will be like:


  </Step>
  <Step title="Start the serverless LLM function">
  ```sh
  yomo run
  ```
  <Note>Press `Ctrl-C` to stop the Serverless LLM Function.</Note>

  </Step>
