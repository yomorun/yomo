# YoMo

YoMo is an open source LLM Function Calling Serverless Framework for building reliable and low-latency AI applications.

There are two main components in YoMo:

- **LLM Providers**: Provides OpenAI API compitable interface for all models, currently support:
  - OpenAI
  - Azure OpenAI
  - Llama
  - Mistral
  - Ollama
- **Function Calling Serverless**: Write once, run on any model.

## Getting Started

<Steps>
### Install CLI

```bash copy
curl -fsSL "https://get.yomo.run" | sh
```

### Initialize a StreamFunction project

```bash copy
yomo init llm-tool-get-time
```

### Build 

```bash {3}
ᐅ cd llm-tool-get-time

ᐅ yomo build app.go
```
