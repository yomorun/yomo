# OpenAI Provider

[OpenAI](https://openai.com) 

## Configuration

```yaml
name: ai-zipper
host: 0.0.0.0
port: 9000

bridge:
  ai:
    server:
      addr: localhost:9000
      provider: openai
      
    providers:
      openai:
        api_key: sk-xxxxxxxxxxxxxxxxx
        model: chatgpt-4o-latest
```

## How to get API Key

Create an API key in [your OpenAI dashboard](https://platform.openai.com/api-keys).

## Support Models

- `gpt-4-turbo`
- `gpt-4`
- `gpt-4o`
- `chatgpt-4o-latest`: https://platform.openai.com/docs/models/gpt-4o
- `gpt-4o-mini`
- `gpt-3.5-turbo`
