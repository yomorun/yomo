# LLM Bridge

LLM Bridge provides an OpenAI API-compatible HTTP server and implements the [Chat API](https://platform.openai.com/docs/api-reference/chat/create). It is extended with a powerful framework to write [Function Calling](https://platform.openai.com/docs/guides/function-calling) in a strongly-typed language and can be deployed as a stateful serverless application.
